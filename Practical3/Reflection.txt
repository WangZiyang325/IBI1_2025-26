Reflection on Git and GitHub
This Git and GitHub course was a valuable learning journey for me.
I started by registering a GitHub account and learning basic commands like cloning and committing files. 
I faced many small issues at first, such as confusing terminal commands and file path errors.

The biggest challenge was pushing code to GitHub. I had to use a VPN to fix network problems, and I spent time setting up SSH keys to replace password authentication.
After careful steps, I finally managed to push my work successfully.

Later, I created my personal github.io repository and enabled GitHub Pages. When I opened my own webpage at https://wangziyang325.github.io, I felt a strong sense of achievement.

This experience taught me to be patient with technical learning. Every mistake helped me grow, and I am now more confident using Git and GitHub for future studies.
